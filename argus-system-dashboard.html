System Overview

Purpose

ARGUS 5.0 is a comprehensive business operations management system designed for Clear Cut Methods. It provides secure, role-based access to CRM, project management, task tracking, and consultation management.

Key Capabilities

  • Role-based access control with 5 user types
  • Complete consultation lifecycle management
  • 360° client relationship tracking
  • Visual project management with Kanban
  • Advanced task management with priorities

Security Implementation

Authentication Flow

1. User accesses system
2. getUserData(email) validation
3. canUserAccessSystem(userId) check
4. Role-based data filtering
5. Secure session establishment

Data Access Pattern

getProjects() {
  // Check user role
  if (FULL_ACCESS_ROLES.includes(role)) {
    return allProjects;
  } else if (PROJECT_ROLES.includes(role)) {
    return userProjects;
  }
  return [];
}
